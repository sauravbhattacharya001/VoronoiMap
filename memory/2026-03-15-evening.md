## Gardener/Builder Migration: Task Scheduler → Cron
- **Problem:** Both scripts used `--agent main` which routed everything to main session, blocking Shubho's chat
- **Attempted fix 1:** Added `--session-id gardener/builder` — didn't work, still hit main
- **Attempted fix 2:** Removed `--agent main`, kept `--session-id` — still didn't work
- **Root cause:** `openclaw agent` CLI always routes to main session regardless of `--session-id` when no `--to` is specified
- **Solution:** Disabled Windows Task Scheduler tasks, created OpenClaw cron jobs with `sessionTarget: "isolated"`
- Cron job IDs:
  - Gardener: 527d5247-d94d-41df-8193-bad3b2ed597c
  - Builder: 2e530296-a6d8-4ac1-8b1b-77ef34f79285
- Both set to every 30 min, 30 min timeout, delivery mode "announce"
- **Working perfectly** — isolated sessions, results announced to main
