# 🧠 Zalenix Memory

Private memory backup repository for the **Zalenix** AI assistant.

This repo stores persistent memory files, identity documents, and workspace context that Zalenix uses across sessions to maintain continuity, personality, and long-term knowledge.

## ⚠️ Private Repository

This repository is intended for internal use only. It contains personal context and workspace data for the Zalenix AI agent.

## Structure

- **Memory files** — Daily logs and long-term memory (`MEMORY.md`, `memory/`)
- **Identity** — Soul, identity, and personality configuration (`SOUL.md`, `IDENTITY.md`)
- **Projects** — Active project context and workspace state
- **Heartbeat** — Periodic task configuration
